npx

npx 是什么

npx 会帮你执行依赖包里的二进制文件。可以运行没有安装的npm服务。

npx 会自动查找当前依赖包中的可执行文件，如果找不到，就会去 PATH 里找。如果依然找不到，就会帮你安装！

例如：npx http-server 可以一句话帮你开启一个静态服务器
